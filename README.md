## Circular

Install [NodeJS](https://nodejs.org/), [GIT](http://git-scm.com/) and NPM.

Install `http-server` to quickly init local server
```shell
npm install http-server -g
```

Git repository
```shell
git clone git@github.com:felquis/circular.git

cd circular/
```

Init server
```
http-server -o
```
